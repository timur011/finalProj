Functions

In Javascript, functions are code designed to perform a certain task. They can be designed to perform pretty much every task, and make the code a lot more organized and reusable.


Lets say I want to make a function that takes in the users input as an age and returns it to the user. I will call it returnAge().

I will do this by asking the user for input (using the built in "prompt" function) then changing the innerHTML of a p element to the users input!
